# Quake-3-Arena-Anime-Bubble-Gum - Custom Modern Q3 Game Build

My own build of Quake 3 Arena: Anime bubble gum with a fully configured config and other tricks, more than a hundred standard commands are configured, and more than a hundred new ones from modern osp mods, in my cfg, a modern modified version of the quake game client with improved modern engines, textures, models, menus, sounds, upgrades of osp mods, for 2-4k resolutions, 250-300+ Hz monitors, 250+ FPS, with movement scripts in CFG, used even with magnetic keyboards and mice at 8000 Hz, all builds play differently depending on engines, mods and configs, my build has undergone strict tuning - so that it feels no different from stock quake, only better, builds are played on Moscow q3msk servers, naturally with the sv_pure 0 command on the servers they supports loading custom models in the client. Launch it via a batch file named "best launcher" - open it with Notepad and set your monitor resolution in all bat files. You can also set in_mouse 1 in the game itself If the mouse feels unusual because I have special controls. My controls and config are in the video. If you want, you can install third-party mods like excessive, efreeze, defrag, and others, but you'll have to configure them yourself. If you want to upload your own configs, you'll also have to configure everything yourself.
