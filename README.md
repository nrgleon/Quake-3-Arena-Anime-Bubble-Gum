My own build of Quake 3 Arena: Anime bubble gum with a fully configured config, Modern modified version of the quake 3 game client with improved modern engines, textures, models, menus, sounds, upgrades of osp mods, my build has undergone strict tuning - so that it feels no different from stock quake, only better.
